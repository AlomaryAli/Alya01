- `by-lj-logo.png` - Brand logo (recommended: 500x500px, transparent background)

### لذة السبايسي (Lathat Spicy)

- `lathat-logo.png` - Brand logo (recommended: 500x500px, transparent background)

### Third Brand
- `third-brand-hero.jpg` - Hero background image (recommended: 1920x1080px)
- `third-brand-logo.png` - Brand logo (recommended: 500x500px, transparent background)

## Optional:
- `favicon.ico` - Website favicon
- `og-image.jpg` - Open Graph image for social sharing (1200x630px)

## Image Guidelines:
- Use WebP format for better performance
- Optimize images before uploading
- Use descriptive file names
- Keep file sizes under 500KB for hero images
- Use PNG for logos with transparency
- Maintain aspect ratios
